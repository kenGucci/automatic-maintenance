API Reference

Everything you need to integrate AutoMend into your monitoring stack.

Overview Dashboard Agents Pro API

Overview

All endpoints return JSON. The base URL is your deployed AutoMend dashboard URL (e.g. https://automatic-maintenance.vercel.app). Data is refreshed every 5 minutes.

Dashboard API

GET /api/overview

System health overview — health score, uptime, tasks, and alert counts.

{ "agent_status": "running", "system_health_score": 91.3, "uptime": "96h 43m", "total_alerts": 14, "critical_alerts": 1, "tasks_completed_today": 11, "tasks_pending": 3, "last_check": "2026-06-08T12:00:00" }

GET /api/metrics

Current system metrics — CPU, memory, disk, network, and process count.

{ "cpu_usage": 42.5, "memory_usage": 67.3, "disk_usage": 34.1, "network_in_mbps": 12.7, "network_out_mbps": 8.2, "active_processes": 186, "timestamp": "2026-06-08T12:00:00" }

GET /api/metrics/history

24-hour metric history (hourly snapshots) for sparkline charts.

GET /api/alerts

Recent alerts and events — critical, warning, and info severity.

GET /api/tasks

Maintenance task history — names, statuses, durations, and timestamps.

GET /api/diagnostics

Service health diagnostics — status, latency, and recommendations.

{ "services": [ {"name": "API Gateway", "status": "healthy", "latency_ms": 48}, {"name": "Database", "status": "healthy", "latency_ms": 79} ], "overall_status": "operational", "recommendations": [ "Consider increasing cache TTL for frequently accessed endpoints" ] }

GET /api/settings

Current agent configuration — thresholds, intervals, and notification channels.

POST /api/settings

Update configuration settings. Accepts JSON body.

Agent API

GET /api/agents

Multi-agent fleet overview — health, tasks, success rate, and runtime for all CrewAI agents.

{ "agents": [ { "name": "Production-Web-01", "role": "Web Server", "status": "active", "health_score": 92, "tasks_completed": 187, "success_rate": 96, "cpu": 52, "memory": 68, "runtime": "72h 15m" } ], "crew_status": "operational", "total_tasks_completed": 887, "average_success_rate": 93.6 }

Pro API

Pro Dashboard endpoints for advanced fleet management. Available in the Pro Dashboard.

GET /api/pro/agents

Pro fleet agents with detailed metrics (status, health, CPU, memory, alerts).

GET /api/pro/alerts

Cross-agent alert feed with agent name and severity.

GET /api/pro/analytics

Aggregated analytics — response time, maintenance tasks, remediation rate, and history.

GET /api/pro/maintenance

Per-agent maintenance schedules with last-run timestamps and status.

POST /api/pro/connect

Connect a new agent to the fleet. Returns a new agent ID.
