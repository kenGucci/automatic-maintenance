# Changelog

All notable changes to AutoMend are documented here.

## 1.0.0 (2026-06-08)

### Added
- Autonomous system monitoring agent (CPU, memory, disk, network)
- Intelligent diagnostic engine with root cause analysis
- Auto-remediation with configurable safety boundaries (dry-run, approval, auto)
- CrewAI multi-agent orchestration (Senior Researcher, Senior Coder, Code Reviewer)
- Real-time Flask dashboard with live metrics and charts
- Pro Dashboard with multi-agent fleet management
- REST API with 10+ endpoints for monitoring and configuration
- Landing page with features, architecture, tech stack, and pricing
- GitHub Actions CI/CD pipeline (daily maintenance at 3 AM UTC)
- Deterministic mock data (5-minute time-bucketed windows)

### Infrastructure
- Vercel serverless deployment (Python Flask via api/index.py)
- Open source under ISC license
