# Contributing to Automatic Maintenance

First off, thank you for considering contributing! Here's how you can help:

## Reporting Bugs

- Use a clear and descriptive title
- Describe the exact steps to reproduce the problem
- Explain the expected behavior and what happened instead
- Include your environment details (Node version, OS, etc.)

## Suggesting Enhancements

- Use a clear and descriptive title
- Provide a step-by-step description of the suggested enhancement
- Explain why this enhancement would be useful

## Pull Requests

- Follow the JavaScript style guide
- Include appropriate test cases
- Ensure all tests pass: `npm test`
- Format code: `npm run format`
- Update documentation as needed

## Development Setup

```bash
git clone https://github.com/YOUR-USERNAME/automatic-maintenance.git
cd automatic-maintenance
npm install
npm run dev
```

## Scripts

```bash
npm start          # Start the agent
npm run dev        # Development mode with auto-reload
npm test           # Run tests
npm run lint       # Check code style
npm run format     # Format code
```

Thank you for contributing! 🚀
